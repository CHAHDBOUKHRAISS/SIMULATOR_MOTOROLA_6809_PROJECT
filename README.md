# Motorola 6809 Microprocessor Simulator

## Overview
This repository contains an academic research project focused on the design and implementation
of a Motorola 6809 microprocessor simulator.

The project demonstrates competencies in:
- low-level system modeling
- instruction set emulation
- debugging tool design
- software architecture using Java

It is intended both as an educational platform and as a technical showcase.

---

## Project Objectives
- Emulate the complete Motorola 6809 instruction set
- Provide a controllable execution environment for testing and learning
- Offer debugging and visualization tools to inspect processor behavior
- Model realistic processor state transitions and memory interactions

---

## Core Features

### Instruction Set Emulation
- Full support for Motorola 6809 instructions
- Accurate handling of addressing modes and processor operations

### Processor State & Registers
- Real-time tracking of all registers and flags
- Manual modification of processor state for experimentation

### Integrated Debugger
- Breakpoints
- Step-by-step execution
- Instruction tracing

### Memory Inspection & Visualization
- Live memory inspection
- Memory editing during execution
- Visual memory representation

### Input / Output Simulation
- User input simulation
- Interrupt signal generation
- Output routing to console or graphical interface

### Graphical User Interface
- Intuitive GUI for monitoring execution
- Visual representation of processor internals
